Теоретическая часть:

    1. Перечислите основные единицы измерения в CSS.
    Разделите их на группы:
    1) Абсолютные единицы
    2) Относительные единицы

    Ответ:
        1) Абсолютные единицы:
           px - пиксели
           cm - сантиметры
           mm - миллиметры
           in - дюймы
           pt - пункты
           pc - пайки (picas) 

        2) Относительные единицы:
           % - процент
           em - единица, высчитываемая относительно размера шрифта родителя
           rem - единица, высчитываемая относительно размера шрифта корневого элемента(html)
           vw - процент от ширины окна браузера
           vh - процент от высоты окна браузера
           vmin - процент от меньшей стороны окна
           vmax - процент от большей стороны окна
           ch - ширина символа "0"
           ex - относительная единица измерения, равная текущей высоте строчной буквы x(x-height) для используемого шрифта

    2. Объясните разницу между относительными и абсолютными единицами измерения.
    Приведите примеры, когда стоит использовать те или иные единицы 
    (например, px для фиксированных элементов и em для масштабируемых текстов)

    Ответ:
        1) Разница между относительными и абсолютными единицами измерения заключается в том,
        что абсолютные единицы имеют фиксированное значение и не изменяются в зависимости от размеров экрана или родительских элементов,
        в свою очередь относительные единицы зависят от контекста:
        размера шрифта, размеров родителя, размеров окна браузера.

        2) Примеры использования:
           Абсолютные единицы:
           px - используется для фиксированных размеров элементов, например border, button
           cm - используется при подготовке печатных документов
           mm - используется в печатных стилях
           in - используется для физических размеров при печати
           pt - используется для задания размеров шрифта в печатных документах
           pc - используется в типографике и печати

           Относительные единицы:
           % - используется для адаптивной ширины и высоты элементов
           em - используется для масштабируемых отступов и текста относительно размера шрифта родительского элемента
           rem - используется для глобальной типографики относительно корневого элемента
           vw - используется для размеров относительно ширины окна браузера
           vh - используется для размеров относительно высоты окна браузера
           vmin - используется для адаптации под меньшую сторону экрана
           vmax - используется для адаптации под большую сторону экрана
           ch - используется для ограничения ширины текста по количеству символов
           ex - используется относительно высоты символа x, применяется редко

    3. Опишите, как в CSS происходит определение углов
    и какие единицы измерения углов существуют
    (например: deg, rad, turn, grad)

    Ответ:
        1) В CSS определение углов происходит при помощи типа данных angle:
           <angle> представляет собой значение угла, выраженное в градусах, градах, радианах или оборотах. 
           Он используется, например, в <gradient> и в некоторых функциях transform.
           Состоит из числа (<number>), за которым следует одна из единиц измерения, представленных ниже. 
           Между литералом единицы измерения и цифрой нет пробела. 
           После значения 0 указывать единицу измерения необязательно.

        2) Существуют следующие единицы измерения углов:
           deg - представляет угол в градусах. Один полный круг равен 360deg. Например: 0deg, 90deg, 14.23deg.
           rad - представляет угол в радианах. Один полный круг равен 2π или примерно 6.2832rad. 1rad - это 180/π градусов. Например: 0rad, 1.0708rad, 6.2832rad.
           turn - представляет угол в количестве оборотов. Один полный круг равен 1turn. Например: 0turn, 0.25turn, 1.2turn.
           grad - представляет угол в градах. Один полный круг равен 400grad. Например: 0grad, 100grad, 38.8grad.
